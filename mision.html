Misión

Nuestra misión es la de compartir Información de manera clara, sencilla y estructurada, de tal manera que el usuario pueda entender lo que esta viviendo. De tal forma que todo el contenido que se muestra esta recopilado de diferentes páginas o sitios que se dedican a publicar información sobre este gran mundo del Cine y por lo tanto nos dedicamos a recopilar toda esa información para que las personas que quiera buscar alguna película o serie que se estrenó o este a estrenarse pueda visitar nuestro sitio y revisar su ficha técnica, trailer, imágenes, opiniones de expertos y del público en general.
De esta manera se pretende que la persona cuando acabe de revisar la información este preparado para tomar una decision.

Visión

Nuestra visión dentro de 5 meses es tener un sitio web totalmente funcional con todas las herramientas necesarias para dar un mejor servicio y el de seguir compartiendo información de manera clara, estructurada, en tiempo y forma para todo tipo de persona de manera gratuita si ningún costo.
Además de tener un total de 10 usuarios suscrito a nuestro sitio para poder enviarle información del nuevo contenido que se estará agregando.
